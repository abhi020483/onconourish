# Gastric Cancer Nutrition Rules — source reference

This is the client-supplied source (ESPEN/ESMO + ICMR-NIN aligned) that the gastric
module is built from. The machine-evaluable, engine-wired version lives in
`tool/gastric.js` (editable config: rule sets, NRS-2002, route algorithm). This file
is kept for provenance and clinical-advisor reference.

Sources: ESPEN Practical Guideline — Clinical Nutrition in Cancer (2021); ESPEN Clinical
Nutrition in Surgery (2021/2025); ESMO Gastric Cancer living guideline (supportive care
& nutrition); upper-GI / gastric nutrition reviews (2016–2023); ICMR-NIN Dietary
Guidelines for Indians (2024).
